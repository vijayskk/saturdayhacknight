POST /predict_account/

Predicts whether a social media account is fake or not.

Request Parameters

Parameter Description
username The username of the social media account.
fullname The full name of the account owner.
profile_picture Whether there is a profile picture (0 for no, 1 for yes).
description The description of the account.
external_link Whether there is an external link or URL in the description (0 for no, 1 for yes).

Response

The response will indicate whether the account is fake or not.

If the account is fake, the response will be:

{"result": "The account is fake"}

If the account is not fake, the response will be:

{"result": "The account is NOT fake"}

POST /create_user/

Creates a new user account.

Request Parameters

Parameter Description
username The username of the new user.
password The password of the new user.

Response

The response will indicate the success or failure of the user creation.

If the user is created successfully, the response will be:

{"message": "User created successfully"}

If there is an error during user creation, the response will include an error message.

GET /user/{id}/

Retrieves details of a specific user.

Path Parameters

Parameter Description
id The unique identifier of the user.

Response

The response will include details of the user.

Example response:

{"username": "example_user", "email": "example@example.com", "age": 30}
